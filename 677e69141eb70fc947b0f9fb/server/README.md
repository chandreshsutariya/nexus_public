# CoolMatch Application

CoolMatch is a comprehensive dating platform designed for interactive one-on-one chatting and broadcasting for users, particularly tailored for female broadcasters and male users looking for engaging social experiences.

## Project Structure

This project consists of a frontend React Native application and a backend Node.js server.

### Client (Frontend)

- **Main Technologies**: React Native, Redux, Firebase
- **Key Folders**:
  - `components/`: Reusable UI components.
  - `pages/`: Different pages of the application.
  - `services/`: API service functions.
  - `assets/`: Images and other static assets.
  
### Server (Backend)

- **Main Technologies**: Node.js, Express.js, MongoDB
- **Key Folders**:
  - `controllers/`: Business logic for user and authentication operations.
  - `models/`: Database models for users and other entities.
  - `routes/`: API endpoint definitions for user and authentication actions.

## Setup Instructions

1. **Backend Setup**:
   - Navigate to the `server` directory.
   - Run `npm install` to install dependencies.
   - Configure environment variables in `.env`.
   - Start the server with `nodemon server.js`.

2. **Frontend Setup**:
   - Navigate to the `client` directory.
   - Install frontend dependencies using `npm install`.
   - Run the React Native app with `npx react-native run-android` or `npx react-native run-ios`.

## Features

- One-on-one matching system with coin-based interactions.
- Subscription model for premium content.
- Private and secure calling options with built-in safety features.
- AI-assisted messaging and notifications to enhance user engagement.

## Future Improvements

- Enhance user experience with additional features based on user feedback.
- Expand marketing strategies to attract a wider user base.
- Implement robust data protection measures to ensure user privacy and security.
