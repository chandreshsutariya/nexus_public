# CoolMatch Project Overview

## Introduction
CoolMatch is a comprehensive dating platform designed to facilitate one-on-one interactions between users. The application provides a unique environment for female broadcasters to engage in live chats, earn money, and meet new people, while providing male users with opportunities to interact with female members.

## Objectives
- Create an interactive platform aimed at enhancing social experiences.
- Provide financial compensation for female broadcasters.
- Foster confidence and improve social interactions among female users.

## Features
- **Matching System**: Male users can swipe to engage in conversations. Coin deductions apply based on the matching category.
- **Private Call System**: Male users can privately call broadcasters, with coins deducted per time segment.
- **Subscription and Special Wall**: Male users can subscribe to access exclusive content shared by female members.
- **Security Measures**: Various measures, including content filtering and user verification, are in place to ensure user safety and compliance with rules.

## Target Audience
- Female Broadcasters: Seeking to monetize their broadcasting efforts.
- Male Users: Interested in connecting and interacting with female members.

## Technology Stack
- **Frontend**: React Native for the mobile application.
- **Backend**: Node.js with Express for the server application.
- **Database**: MongoDB for data management.

## Installation Instructions

### Backend Setup
1. Clone the repository and navigate to the `server` directory.
2. Install dependencies: 
   ```bash
   npm install
   ```
3. Set up your MongoDB connection string in the `.env` file.
4. Start the server:
   ```bash
   node server.js
   ```

### Frontend Setup
1. Navigate to the `client` directory.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the React app:
   ```bash
   npm start
   ```
